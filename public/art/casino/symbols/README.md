# カジノ機能 絵柄アセット

スロット機種「賢者の石を求めて」で使用する絵柄画像。

## ファイル一覧

| ファイル | 絵柄 | 用途 |
|----------|------|------|
| `bell.png` | ベル（黄銅片） | 最高頻度小役 |
| `watermelon.png` | スイカ（月光草） | レア小役 |
| `cherry.png` | チェリー（血色結晶） | 角/中段チェリー |
| `replay.png` | リプレイ（賢者の石の欠片） | 再遊技 |
| `big7_red.png` | 赤7（火竜の鱗） | BIG揃い・REG異色揃い用 |
| `blue7.png` | 青7（水竜の鱗） | REG異色揃い・青青青（ART契機）用 |

## 備考

- 参考画像をそのまま使用。加工はしない
- チャンス目・リーチ目は専用絵柄を持たず、既存絵柄の停止パターンで表現（`src/game/casino/data/chancePatterns.js`）
- 表示サイズはランタイム側のCSSで統一（各画像の原サイズは維持）

## 出典

原画は `docs/casino/参考アセット/` から複製（プロジェクト内部の参考素材）。
